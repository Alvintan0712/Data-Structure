2.  假设 $n$ 为 $2$ 的乘幂 ， 并且 $n>2$ ， 试求下列算法的时间复杂度及变量 `count` 的值（以 $n$ 的函数形式表示 ， 在本课程作业和考试范围内 ， 除非特别声明 ， 对数都是以 $2$ 为底 ， 即 $\log_2n$ 可表示为 ： $\log n$ 或者 $log(n)$；$m*n$ 的形式可表示为 $mn$ ）。

   ```C
   int time (int n) {
   	int count=0, x=2;
       while ( x < n/2 ) {
   		x=2*x;
   		count++;
   	}
   	return (count);
   }    
   ```

   **Ans:时间复杂度：$O(\log n)$	`count` 值 ：  $\log n-2$ **
